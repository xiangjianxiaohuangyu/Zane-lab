## 项目特点
- **可阅读** - 优雅的排版，舒适的阅读体验
- **可扩展** - 添加内容无需修改代码
- **可维护** - 清晰的结构，类型安全
- **可持续** - 长期演进，自然生长

## 技术栈
- **构建工具**: Vite 6.x
- **框架**: React 18.x + TypeScript 5.x
- **路由**: React Router DOM 6.x
- **样式**: Tailwind CSS 3.x
- **Markdown 处理**: gray-matter + remark
- **设计风格**: 玻璃拟态（Glassmorphism）

## 核心功能
1. **项目展示** - 展示个人开发项目
2. **写作空间** - 随笔、年终总结、小说创作
3. **记录系统** - 电影、书籍、游戏、音乐收藏
